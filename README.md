#  IT Support Portfolio

This repository showcases my practical skills in IT support and troubleshooting. It includes common technical issues, their causes, and step-by-step solutions based on real-world scenarios.

##  1. Computer Not Connecting to Internet

**Possible Causes:**

* Network cable unplugged
* Incorrect IP configuration
* Faulty router or switch

**Solution Steps:**

1. Check physical connections (Ethernet/Wi-Fi)
2. Run `ipconfig /all` to check IP address
3. Renew IP using `ipconfig /renew`
4. Restart router and computer
5. Test connectivity using `ping 8.8.8.8`

##  2. Slow Computer Performance

**Possible Causes:**

* Too many startup programs
* Low system memory (RAM)
* Malware or viruses

**Solution Steps:**

1. Disable unnecessary startup programs via Task Manager
2. Run a full antivirus scan
3. Delete temporary files using Disk Cleanup
4. Upgrade RAM if necessary

##  3. Printer Not Responding

**Possible Causes:**

* Printer offline
* Driver issues
* Network connectivity problems

**Solution Steps:**

1. Check printer power and connection
2. Set printer as default
3. Restart Print Spooler service
4. Reinstall or update printer drivers

##  4. User Cannot Log In

**Possible Causes:**

* Incorrect password
* Account locked
* Expired credentials

**Solution Steps:**

1. Verify login credentials
2. Reset password
3. Unlock account in system settings
4. Check domain connection (for enterprise systems)

##  5. No Network Access (Limited Connectivity)

**Possible Causes:**

* DHCP server issues
* Incorrect network settings
* Faulty network adapter

**Solution Steps:**

1. Restart network adapter
2. Run Windows Network Troubleshooter
3. Assign IP manually (if needed)
4. Update network drivers
